# Security Policy

Recallant is currently a documentation-first project. The active security model and implementation requirements live in [docs/SECURITY.md](docs/SECURITY.md).

Do not open public issues containing secrets, tokens, private server paths, raw memory exports, database dumps, or sensitive project history.

For now, report security concerns privately to the project owner. A public disclosure process can be added before Recallant is promoted beyond early development.
